The Window Object

  • The Browser Object Model
    • The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen.
    • The BOM only makes sense in a browser environment.
    • In a browser environment, the global object is the window object.

  • Browser Information
    • Which Browser?
      • window.navigator.userAgent
    • Location, Location, Location
      • The window.location property is an object that contains information about the URL of the current page.
      • window.location.href

  • The Browser History
    • The window.history property can be used to access information about any previously visited pages in the current browser session.
      • window.history.length
        window.history.go()
        window.history.forward()
        window.history.back()

  • Controlling Windows
    • A new window can be opened using the window.open() method.
      • const popup = window.open('https://sitepoint.com','SitePoint','width=400,height=400,resizable=yes');
        popup.close();
        window.moveTo(0,0);
        window.resizeTo(600,400);
    • Screen Information
      • The window.screen object contains information about the screen the browser is displayed on.
    • The Document Object
      • Each window object contains a document object.
      • document.write()

  • Cookies
    • Cookies are small files that are saved locally on a user’s computer.
    • Creating Cookies
      • document.cookie = 'name=Superman';
    • Changing Cookie Values
      • A cookie’s value can be changed by reassigning it to document.cookie using the same name but a different value.
      • document.cookie = 'city=Gotham'
    • Reading Cookies
      • To see the current contents of the cookie jar, simply enter
      • document.cookie
    • Cookie Expiry Dates
      • Cookies are session cookies by default.
      • document.cookie = 'name=Batman; max-age=86400' // 86400 secs = 1 day
    • The Path and Domain of Cookies
      • document.cookie = 'name=Batman; path=/'
        document.cookie = 'name=Batman; domain=sitepoint.com';
    • Secure Cookies
      • document.cookie = 'name=Batman; secure';
    • Deleting Cookies
      • To remove a cookie, you need to set it to expire at a time in the past:
      • document.cookie = 'name=Batman; expires=Thu, 01 Jan 1970 00:00:01 GMT'

  • Timing Functions
    • The window object provides some useful methods for scheduling the execution of a function, and for repeatedly executing functions at regular intervals.
    • setTimeout()
      • The window.setTimeout() method accepts a callback to a function as its first parameter and a number of milliseconds as its second parameter.
      • window.setTimeout( () => alert("Time's Up!"), 3000);
    • setInterval()
      • It will repeatedly invoke the callback function after every given number of milliseconds.
      • const summon = window.setInterval(chant,1000);
    • Animation
      • The setTimeOut() and setInterval() methods can be used to animate elements on a web page.
