Document Object Model

  • Introduction to the DOM
    • DOM represents an HTML document as a network of connected nodes that form a tree-like structure.
    • The DOM treats everything on a web page as a node.
    • The DOM is not actually part of JavaScript because it is language agnostic.
    • We can use JavaScript to access and modify different parts of a web page using a special built-in object called document.

  • Getting Elements
    • The DOM provides several methods that allow us to access any element on a page.
    • These methods will return a node object or a node list, which is an array-like object.
    • Legacy DOM Shortcut Methods
      • Document.body returns the body element of a web page
      • Document.images returns a node list of all the images contained in the document.
      • Document.links returns a node list of all the <a> elements and <area> elements that have an href attribute.
      • Document.anchors returns a node list of all the <a> elements that have a name attribute.
      • Document.forms returns a node list of all the forms in the document.
    • Node lists are array-like objects, but they are not arrays.
    • getElementById()
      • It returns a reference to the element with a unique id attribute that is given as an argument.
      • const h1 = document.getElementById('title');
    • getElementsByTagName()
      • It return a live node list of all the elements with the tag name that is provided as an argument.
      • const listItems = document.getElementsByTagName('li');
    • getElementsByClassName()
      • It return a live node list of all elements that have the class name that is supplied as an argument.
      • const heroes = document.getElementsByClassName('hero');
    • document.querySelector()
      • Allows you to use CSS notation to find the first element in the document that matches that matches a CSS selector provided as an argument.
    • document.querySelectorAll()
      • uses CSS notation but returns a node list of all the elements in the document that match the CSS query selector.

  • jQuery
    • jQuery is a popular JavaScript framework that makes it very easy to find elements on a page using a CSS-style syntax.
    • It uses document.querySelectorAll() in the background whenever it can.

  • Navigating the DOM Tree
    • Node objects have a number of properties and methods for navigating around the document tree.
    • The childNodes property is a list of all the nodes that are children of the node concerned.
    • The childNodes property returns all the nodes that are children of an element.
    • The firstChild property returns the first child of a node.
    • The lastChild property returns the last child of a node
    • The parentNode property returns the parent node of an element.
    • The nextSibling property returns the next adjacent node of the same parent.
    • The previousSibling property returns the previous adjacent node.
    • The textContent property returns the text contained within an element.

  • Getting and Setting Attributes
    • Setting An Element’s Attributes
      • The setAttribute can change the value of an element’s attributes.
      • wonderWoman.setAttribute('class', 'villain');
      • If an element does not have an attribute, the setAttribute method can be used to add it to the element.
      • wonderWoman.setAttribute('id','amazon');
      • The add method can be used to add a class to an element without overwriting any classes that already exist.
      • wonderWoman.classList.add('warrior');
      • The remove method will remove a specific class from an element.
      • wonderWoman.classList.remove('warrior');
      • The toggle method is a particularly useful method that will add a class if an element doesn’t have it already, and remove the class if it does have it.It returns true if the class was added and false if it was removed.
      • wonderWoman.classList.toggle('hero');
      • The contains method will check to see if an element has a particular class.
      • wonderWoman.classList.contains('hero');

  • Creating Dynamic Markup
    • The process to follow each time you want to create a new element with text content is this:
      • Create the element node.
      • const flash = document.createElement('li');
      • Create the text node.
      • const flashText = document.createTextNode('Flash');
      • Append the text node to the element node.
      • flash.appendChild(flashText);
    • This can be made simpler by using the textContent property that every element object has. This will add a text node to an element without the need to append it.
      • const flash = document.createElement('li');
      • flash.textContent = 'Flash';

  • Updating CSS
    • Any CSS property names that are separated by dashes must be written in camelCase notation, so the dash is removed and the next letter is capitalized because dashes are not legal characters in property names.
      • the CSS property background-color becomes backgroundColor.
      • superman.style.backgroundColor = 'blue';
    • One particularly useful CSS property often employed is the display property.
      • You can hide the superman element with the following code:
      • superman.style.display = 'none';
      • The element can be made to 'reappear' by changing the display property back to block:
      • superman.style.display = 'block';
